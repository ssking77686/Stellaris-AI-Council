# 银河帝国指挥中心 v1.0.0

首个公开发布版本。

## 本版本包含

- 完整的前端 UI（6 个功能页面）
- 内置模拟帝国数据，无后端也可完整预览
- Windows 桌面应用（exe 单文件，双击即用）
- 静态网页版（可用浏览器打开）

## 使用说明

### Windows 桌面应用

双击 `银河帝国指挥中心.exe` 启动。首次如遇 SmartScreen 警告，点击「更多信息」→「仍要运行」。

- Windows 10/11 无需安装任何依赖
- 关闭窗口即退出，不驻留后台

### 浏览器版

在本目录启动本地服务器后访问：

```bash
npx serve .        # Node.js
python -m http.server 8080   # Python
```

> 不能直接双击 index.html，需通过 HTTP 服务器打开。

## 已知问题

- exe 未数字签名，Windows 会弹出安全警告
- 部分杀毒软件可能误报（PyInstaller 打包）
- 无后端时对话和朝会功能不可交互
- Orbitron 字体在中国大陆可能无法加载

## 文件说明

| 文件 | 用途 |
|------|------|
